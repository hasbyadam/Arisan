# Arisan App API
Arisan is a Bookkeeping app tailored specifically to manage “Rotating Savings and Credit Association” (RoSCA) or in Indonesia commonly known as "Arisan".
## Description
This is the API for Arisan App
### Current working features
* SignIn / SignUp
* Google oAuth
* Profile CRUD
* Contact List CRUD
* Arisan CRUD
* Filter & Sort
* Automatic Raffle
## Tech Stack
* JavaScript
* Node.js
* Express
* PosgreSQL
* Sequelize
* Heroku
* Google oAuth
* Facebook oAuth
